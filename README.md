# Dianping Favorites Exporter Local

一个只在本机 Chrome 中运行的大众点评收藏导出扩展。它复用你当前 Chrome 已登录大众点评的会话，从你打开的收藏列表页读取页面可见的商户收藏，并可慢速打开商户详情页补充地址、营业时间、推荐菜等字段。

## 特点

- 不读取浏览器 cookie、`localStorage`、密码或其他站点数据。
- 不上传收藏数据，不调用第三方接口。
- 只声明访问大众点评相关页面。
- 支持收藏列表里混合餐厅、酒店、景点、休闲娱乐、购物、生活服务等不同商户类型。
- 导出 `JSONL` 和 `CSV`，其中 `JSONL` 更适合作为 AI agent 的个人上下文。

## 安装

1. 打开 Chrome 的 `chrome://extensions/`。
2. 开启右上角“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择本目录：`dianping-favorites-exporter-extension`。

## 使用

1. 在 Chrome 打开大众点评“我的收藏”或能看到收藏商户列表的页面。
2. 确认已经登录，并且列表能正常显示。
3. 点击 Chrome 工具栏里的 `Dianping Favorites Exporter Local`。
4. 选择自动滚动次数、最多商户数、请求间隔，以及是否补充详情页字段。“最多翻页数”默认是 100，用来限制列表页里“下一页”的读取上限。
   “补充商户详情页字段”默认关闭；开启后，扩展会逐个访问已收集商户的详情页，用来补充地址、电话、营业时间、推荐菜等字段。
5. 点击“开始导出”，进度会显示在页面右下角。
6. 完成后会下载两个文件：
   - `dianping-favorites-*-complete.jsonl`
   - `dianping-favorites-*-complete.csv`

如果中途点击“停止当前导出”，会下载 `partial` 文件，里面是已经抓到的数据。

## 字段

基础字段包括：商户 ID、商户名称、商户链接、商户类型、城市、商圈、分类、评分、人均价格、地址、标签、原始摘要。

详情页补充字段包括：电话、营业时间、推荐菜、特色、详情补充状态。

`商户类型` 是本地根据分类和页面文本推断出来的粗分类，可能是：

- `restaurant`
- `hotel`
- `attraction`
- `spa`
- `beauty`
- `fitness`
- `entertainment`
- `shopping`
- `service`
- `other`

## 注意

- 推荐先保持“补充商户详情页字段”关闭跑一次，确认收藏列表能完整导出；需要更完整字段时，再开启详情补充并慢速运行。
- 如果页面出现验证码、安全验证或登录失效，扩展会停止或记录失败状态，请在浏览器中人工处理后重试。
- 大众点评页面结构可能变动，导出器使用的是多组选择器和文本兜底，不保证每个字段都能完整命中。
- 不建议长期保留扩展；导出完成后可以在 `chrome://extensions/` 里移除它。

本项目与大众点评没有任何关联，只用于导出本人账号可见的个人收藏数据。
